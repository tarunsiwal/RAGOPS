Introduction to REGOPS APIs

REGOPS offers a variety of APIs to allow developers to interface with REGOPS products and embed them to there websites. We currently provide product-based REST APIs.

REST APIs

Our REST APIs are organized based on REGOPS product, but the base URL for all REGOPS REST API requests is https://regops/api/chatlog/.

API Code

Get a Chat responce GET /api/chatlog 
import requests

url = "https://api.example.com/data"  # Replace with the actual API endpoint URL

try:
    response = requests.get(url)

    # Check if the request was successful (status code 200)
    if response.status_code == 200:
        print("API Response:")
        print(response.json())  # Assuming the response is in JSON format
    else:
        print(f"Error: {response.status_code} - {response.text}")

except requests.RequestException as e:
    print(f"An error occurred: {e}")

Send query to a Chat to get responce POST /api/chatlog 
import requests

url = "https://your-api-endpoint.com/path/to/chatlog/"
# Replace "YOUR_ACCESS_TOKEN" with the actual access token if required for authentication
headers = {
    "Authorization": "Bearer YOUR_ACCESS_TOKEN",
    "Content-Type": "application/json",
}

try:
    response = requests.get(url, headers=headers)
    # Check if the request was successful (status code 200)
    if response.status_code == 200:
        print("API Response:")
        print(response.json())
    else:
        print(f"Error: {response.status_code} - {response.text}")

except Exception as e:
    print(f"An error occurred: {e}")
